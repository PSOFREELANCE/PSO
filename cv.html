Training

Industrial Training at 'CRIS'

Project: E-Rail Ticket
Chanakyapuri, New Delhi
[January 2014 - July 2014]
Overview:

During this 6 month industrial training at the Center For Railways Information Systems (CRIS) in Chanakyapuri, New Delhi, I had the opportunity to work on a real-time project titled "E-Rail Ticket". The aim of this project was develop a comprehensive online ticketing system for railways, leveraging cutting-edge web technologies and database management systems to enhance user experience and operational efficiency.

Project Objective:
  • 1. Steamline Ticket Booking: To create an intuitive platform for users to book train ticket easily.
  • 2. Real-Time Availability: Impliment real-time tracking of seat availability and booking status.
  • 3. User Authentication: Secure user registration and login functionalities.
  • 4. Payment Gateway Integration: Enable secure and multiple payment options for users.
  • 5. Ticket Management: Provide features for booking history, ticket cancellation, and refunds.
Technologies Implemented:
  • 1. JSP (JavaServer Pages): Utilized to create dynamic web pages and manage user interactions.
  • 2. Hibernate: Used for database operations, ensuring seamless communication between the application and Oracle 10 database.
  • 3. HTML & CSS: Employed for structuring and styling the web pages to enhance user interface and experience.
  • 4. Oracle 10: Served as the robust backend database for storing user data, booking details, and transaction records.
  • 5. JavaScript: Incorporated to add interactive elements and improve user engagement on the web application.
